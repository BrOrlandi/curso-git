Git e contribuições para projetos Open Source

Bruno Orlandi

Aprenda desde o controle de versão, ferramentas como Git e GitHub, e como contribuir com projetos Open Source.

Não há nenhum pré-requisito para este curso, você não precisa ser um programador para entender o curso.

Público Alvo

O que aprenderei?

Você aprenderá os fundamentos básicos sobre Controle de Versão e como usar Git. Também aprenderá a contribuir com projetos no GitHub e a trabalhar em equipe usando Git. No começo, aprenderá um pouco sobre Gerência de Configuração. Em seguida, como resolver conflitos e gerenciar branchs.
